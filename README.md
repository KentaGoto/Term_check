# Term_check 
翻訳の用語を簡易チェックするためのツール  

## Usage  
```
> Term_check.pl
Tab-delimited bilingual text file (* .txt): D:\bilingual.txt
Tab-delimited glossary text file (* .txt): D:\Term.txt
```  

**Note**  
対象ファイルおよび用語集は、ソースとターゲットにタブ区切りされ、かつUTF BOMなしで保存されている必要がある
